CASH_REGISTER_README# 🏪 Система управління продажами для розважального центру

Повнофункціональна система касового реєстру з управлінням продажами, клієнтами, чеками та звітами для сімейного розважального центру.

## 📋 Основні можливості

### 📊 Продажі
- **Швидкий вибір послуг** за категоріями:
  - 🎳 Боулінг
  - 🎱 Більярд
  - 🎤 Караоке
  - 🕹️ Ігри та VR
  - 🍹 Бар та закуски

- **Кошик з управлінням**:
  - Додавання/видалення товарів
  - Зміна кількості
  - Автоматичний розрахунок суми
  - Розрахунок податку (10%)

- **Прив'язка клієнта до чека**
- **Автоматична нумерація чеків**

### 👥 Клієнти
- Створення картки клієнта з контактами
- Історія відвідувань
- Відстеження витрат
- Швидкий вибір при продажу
- Останні клієнти в списку

### 🧾 Чеки
- Автоматична нумерація
- Детальна інформація про покупку
- Перегляд та друк чека
- Історія всіх операцій
- Модальне вікно з деталями чека

### 💰 Касова дисципліна
- **Відкриття каси** зі вказанням початкового балансу
- **Закриття каси** з автоматичним створенням Z-звіту
- Відстеження продажів та витрат
- Контроль поточного балансу
- Статистика за зміну

### 🔒 Безпека
- Неможливо створювати чеки при закритій касі
- Всі дані зберігаються локально в localStorage
- Повна історія операцій
- Синхронізація з API (планується)

## 📊 Звіти

### X-Звіт (поточна зміна)
- Статус поточної зміни
- Кількість створених чеків
- Сума продажів
- Поточний баланс каси
- Розбивка по категоріях з прогрес-барами
- Без закриття зміни

### Z-Звіт (закриття зміни)
- Автоматично створюється при закритті каси
- Зберігає всю інформацію про зміну
- Історія всіх завершених змін
- Початковий і кінцевий балансі
- Тривалість зміни
- Касова дисципліна (різниця готівки)
- ТОП-5 найпопулярніших послуг

### Аналітика за період
- 📅 Вибір дати початку та кінця звіту
- 📈 Загальна статистика (дохід, середній чек, кількість клієнтів)
- 📦 Розбивка продажів за категоріями з візуальними індикаторами
- ⭐ ТОП-5 найпопулярніших послуг
- 💰 Відсоток від загального обороту для кожної категорії
- 📊 Щоденна статистика

## 🎨 Візуалізація

- Прогрес-бари для категорій
- Кольорові індикатори для різних метрик
- Зручна навігація між звітами
- Експорт даних для аналізу (планується)
- Друк звітів та чеків

## 🚀 Використання

### Відкриття каси
1. Натисніть кнопку "Відкрити касу" в шапці
2. Введіть початковий баланс
3. Натисніть "Відкрити"

### Додавання товарів
1. Виберіть категорію послуг
2. Натисніть на послугу для додавання в кошик
3. Змініть кількість за необхідності

### Вибір клієнта
1. Виберіть клієнта зі списку останніх
2. Або натисніть "+ Додати клієнта" для створення нової картки

### Оформлення чека
1. Виберіть спосіб оплати (Готівка, Карта, Змішано)
2. Натисніть відповідну кнопку
3. Переглядайте чек в модальному вікні
4. Натисніть "Друк" для друку

### Закриття каси
1. Натисніть кнопку "Закрити касу" в шапці
2. Автоматично створюється Z-звіт
3. Перейдіть на сторінку "Звіти" для перегляду

### Перегляд звітів
1. Перейдіть на сторінку "Звіти"
2. Виберіть тип звіту (X-звіт, Z-звіти, Аналітика)
3. Для аналітики виберіть період дат
4. Натисніть "Друк" для друку звіту

## 💾 Зберігання даних

Всі дані зберігаються локально в `localStorage` браузера:
- Поточна зміна
- Клієнти
- Чеки
- Закриті зміни та Z-звіти
- Послуги

При перезавантаженні сторінки дані автоматично завантажуються.
